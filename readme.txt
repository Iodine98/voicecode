Thanks so much for signing up for the VoiceCode beta!

ACCOUNT SETUP
=============
Please send your github username to ben@voicecode.io so you can be added to the code repository. (If you don't have a github account, it is free to set one up)


SETUP GUIDE
===========
https://github.com/VoiceCode/docs/wiki


TO START VOICECODE
==================

FIRST make sure you are on the most up to date version: (after you are added to the github repo)

cd ~/voicecode ; git pull

THEN

- Option 1
–––––––––––––––––––––––––––––––––––––––––
	./run.sh

- Option 2 (if you want more control)
–––––––––––––––––––––––––––––––––––––––––
	cd ~/voicecode/commando
	meteor run --settings config/settings.json --port 5000


Command Reference (after starting voicecode)
––––––––––––––––––––––––––––––––––––––––––––
http://commando:5000/commands


Getting more info
––––––––––––––––––––––––––––––––––––––––––––

After you are added to the github repo, also check out the page https://github.com/stratogee/voicecode/releases for some extra details/tips/tricks.

Also, after you are added to the github repo, make sure to update VoiceCode to make sure you are on the latest version before you start the program.
