TO START VOICECODE
==================

FIRST make sure you are on the most up to date version:

cd ~/voicecode ; git pull

THEN

- Option 1
–––––––––––––––––––––––––––––––––––––––––
	cd ~/voicecode/commando
	meteor run --settings config/settings.json --port 5000

- Option 2
–––––––––––––––––––––––––––––––––––––––––
	./run.sh
